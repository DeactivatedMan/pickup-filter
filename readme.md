# What the plugin does
In certain scenarios, you may not want to pick up every item you walk over. This plugin lets you set 9 filter profiles, each with 27 whitelist slots, anything not on the whitelist will not be picked up.

# Commands
- `filter` - Opens the main menu with all 9 filters
- `filter off-9` - Switches to a different profile or disables filter
- `filter switch off-9` - Ditto above
- `filter edit 1-9` - Open editor menu for a filter profile
